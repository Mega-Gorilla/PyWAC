# Proposals

改善・リファクタリング提案ドキュメントを格納するディレクトリです。

## 命名規則

```
{番号}-{簡潔な説明}.md
```

例: `003-import-structure-refactoring.md`

## 提案ドキュメントのテンプレート

```markdown
# Proposal #{番号}: {タイトル}

**ステータス**: 提案中 | 承認済み | 実装中 | 完了 | 却下
**関連Issue**: [#{番号}](URL)
**作成日**: YYYY-MM-DD
**対象バージョン**: vX.Y.Z

## 概要
## 背景
## 提案する変更
## 実装計画
## 互換性
## リスク評価
```
